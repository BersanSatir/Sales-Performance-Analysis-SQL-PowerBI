# Sales Performance & Data Integrity Analysis (2003-2005)

## 📌 Project Overview
This project provides a deep dive into historical sales data to evaluate business growth and operational efficiency. A significant portion of the work involved **data cleaning and validation using SQL** to address a "Price Cap" issue that was skewing revenue figures.

## 🛠 Tech Stack
- **SQL Server:** Advanced querying, CASE statements for data validation, and CTEs.
- **Power BI:** Data storytelling through interactive visualizations (Visuals included in the repository).
- **Excel:** Data profiling.

## 🔍 Key Data Insights
- **Financial Growth:** A solid **32.96% YoY growth** identified after correcting price-capped sales data.
- **Market Segment:** The **Medium** deal size category is the most critical segment, driving both volume and value.
- **Seasonality:** Identified a recurring revenue spike in **Q4**, suggesting a need for increased inventory in September.

## 🗂 Project Structure
- `SQL_Scripts/`: Contains the full SQL workflow from cleaning to final analysis.
- `Visuals/`: High-resolution screenshots of the Power BI insights.
- `Analysis_Report.pdf`: Comprehensive professional report with executive summaries.
